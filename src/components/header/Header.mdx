# Header

Header component with different props.

#### Example

<Canvas of={Header.Sample} />

## Usage

```ts
import {Header} from "sld-ui";
import companyLogo from "path/to/companyLogo";
import userLogo from "path/to/userLogo";

const Example = () => {
  return (
      <Header
        userName={"Nome Utente"}
      />
  );
};

export default Example;
```

#### Arguments

- **companyLogo** - A string representing the path to the company logo image.
- **userLogo** - A string representing the path to the user logo image.
- **companyName** - A string representing the name of the company.
- **userName** - A string representing the name of the user.
